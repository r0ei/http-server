# http-server

Simple Multithreaded HTTP Web Server in Python

## Status Codes

_404_ File not found
_200_ OK

## Methods

_GET_

## installation

```bash
$ git clone https://github.com/r0eilevi/http-server
$ cd http-server
$ pip install -r requirements.txt
$ python3 main.py
```
